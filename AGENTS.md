# Project Rules

- 本仓库内的对话、需求说明和文档统一使用中文。
- For this project, verify Lua changes with static logic review only.
- Do not run Lua code, Lua interpreters, or the UrhoX/Urho3D game runtime for validation unless the user explicitly changes this rule.
- Static checks such as reading files, searching references, and reviewing diffs are allowed.
